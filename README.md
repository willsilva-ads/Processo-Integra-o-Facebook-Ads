# Processo de Integração Meta Ads — Nativa Digital

Este é um guia interativo, moderno e dinâmico que orienta o cliente e a equipe no processo de integração e concessão de acesso ao **Meta Business Manager** (Gerenciador de Negócios do Facebook) para a agência **Nativa Digital**.

## 🚀 Funcionalidades

- **Design Premium & Futurista**: Interface escura (Dark Mode) estilizada, com partículas interativas e degradês animados.
- **Menu Lateral Interativo**: Navegação suave e automatizada com indicador de progresso de rolagem (*scroll spy*).
- **Checklist Flutuante de Progresso**: Controle de etapas realizadas com persistência local no navegador (`localStorage`) e animações de conclusão.
- **Textos Editáveis no Navegador**: Capacidade de alterar e personalizar as instruções diretamente na tela e salvar localmente clicando no botão **Salvar** (salvo no `localStorage` do navegador).
- **Cópia Rápida de IDs**: Botão para copiar o ID do Portfólio Empresarial com um clique.
- **Totalmente Responsivo**: Otimizado para visualização perfeita em computadores, tablets e celulares.

## 💻 Como Executar o Projeto

1. Baixe os arquivos do repositório.
2. Abra o arquivo `index.html` em qualquer navegador web de sua preferência (Chrome, Edge, Firefox, Safari, etc.).
3. Não há necessidade de instalar dependências ou rodar servidores locais.
